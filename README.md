## Benchmarking
Each Rust project contains a `input.txt` file, holding the puzzle input. Simply run the project to see the solution appear.
```
# Run Day 1a
cd day01a
cargo run --release

# Benchmark each day
cd execute
cargo run --release --bin bench
```
